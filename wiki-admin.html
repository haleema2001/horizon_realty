Admin Guide

Updating Listings

Open listings.json. Modify price, address, image, or property details.

Adding a New Property

  1. Copy an existing JSON object.
  2. Update id, title, price, category.
  3. Place the new image in the images folder.
  4. Save the file and refresh the site.

Important Notes

No backend database is used. All data is managed through JSON files.
