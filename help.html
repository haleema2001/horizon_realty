Welcome to Horizon Realty Help

This documentation explains the structure, features, and functionality of the Horizon Realty website. Use the sidebar to navigate through different sections including listings, mortgage calculator, JavaScript functionality, and admin features.

Frequently Asked Questions

How does the mortgage calculator work?
The calculator estimates monthly payments using loan amount, interest rate, and term. It applies a standard amortization formula to compute monthly cost.
Are listings dynamically loaded?
Listings are currently static HTML pages grouped by category.
Is this site responsive?
Yes. The layout adjusts for tablets and mobile devices using CSS grid and media queries.
